# Example Repository for bs-node-ext.

# Build

```
npm run build
```

## Usage

Start the server

```sh
yarn build
node src/Demo.bs.js &
```

Use curl to verify it's doing something.

```sh
curl localhost:60023
```

You should see `Hello World`

# Watch

```
npm run watch
```

# Editor

If you use `vscode`, Press `Windows + Shift + B` it will build automatically
